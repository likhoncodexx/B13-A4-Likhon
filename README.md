3. What is Event Bubbling? And how does it work?
Answer : Event Bubbling means a function or  mechanism where an event triggered on a child element propagates upward through its parent and ancestor elements up to the document.


4. What is Event Delegation in JavaScript? Why is it useful?
Answer : It is useful because it improves performance, works for dynamically added elements, and reduces the number of event listeners in your code.
